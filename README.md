Matrix Product

You are given a matrix A and and an integer B, you have to perform scalar multiplication of matrix A with an integer B.

Example Input

Input 1:

A = [[1, 2, 3],
[4, 5, 6],
[7, 8, 9]]
B = 2

Input 2:
A = [[1]]
B = 5

Example Output

Output 1:
[[2, 4, 6],
[8, 10, 12],
[14, 16, 18]]

Output 2:
[[5]]
